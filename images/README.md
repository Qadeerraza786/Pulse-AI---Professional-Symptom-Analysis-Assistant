# Images Directory

This directory contains screenshots and visual assets for the Pulse AI project.

## Required Images

- `welcome-screen.png` - Screenshot of the welcome screen with the input form
- `chat-interface.png` - Screenshot of the chat interface showing an active conversation

## How to Add Images

1. Take screenshots of your application
2. Save them with the exact filenames mentioned above
3. Place them in this `images/` directory
4. The README.md will automatically display them

## Image Guidelines

- **Format**: PNG (recommended) or JPG
- **Size**: Optimize images to reduce file size (recommended max: 1-2MB each)
- **Dimensions**: Maintain aspect ratio, recommended width: 1200-1600px
- **Quality**: Clear, high-quality screenshots that showcase the UI effectively
